# Reviewing system history

In this guide, we will describe how to use the System History tab in the UI to review and audit changes that have been made to systems in the data map.

<Callout emoji="" type="info">To learn how to subscribe to a digest of changes using our System History API, please refer to [this guide](/dev-docs/api/system_history_api).</Callout>

## Exploring system history

To see a digest of system changes, please:

1. Navigate to **Data map** â†’ **View systems**
2. Select the system you want to explore
3. Click on the **History** tab

In the resulting pane, you will find a list of the changes made to the system, along with details about the user who initiated them and the timestamp of each change.

<Screenshot img="/assets/img/tutorials/data-mapping-history-tab.gif" description="Sample annotations" />

To dive into the details of a particular change, click on the entry in the digest to open the change detail panel.
