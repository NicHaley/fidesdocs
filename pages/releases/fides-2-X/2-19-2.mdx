# Fides 2.19.1/2 Release notes
September 12, 2023

These release notes are for Fides 2.19.1 and Fidesplus 2.19.2.

### Features and Enhancements
- System history UI with a modal to examine the details of changes

### Fixes and developer updates
- Re-enabled custom fields for data use declarations
- Fixed issue with saving source and destination systems
- Legal basis for transfer now takes any string as input
